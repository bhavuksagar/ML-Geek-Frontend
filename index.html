Projects

Face Detection

It is a program which detect the face in live video. It use the web camera of the system. It can detect multiple faces in video.It simply create a rectangular box around the faces. This program is developed in Python language. This program can be used in different camera apps and in picture editing applications.

Smile Detector

It is a program which detect the smile on your face in live video. It use the web camera of the system. It simply create a rectangular box around the lips whenever you smile and it is developed in Python language. This program can be used in camera apps for clicking the image of the user whenever he/she smile.

Reviews Analyser

Review analyser helps to understand the sentiment of the review. Amazon used this program on their website to analyse the reveiws of other products. It classify the review into 3 category Positve, Negative and Neutral. It is a NLP(Natural Language Processing) problem and accuracy of the model is 90%. For training we used Airline-Reviews-Dataset which contains thousands of reviews of all category.
